# 📦 Sistema de Gestão de Atacado e Controle de Estoque Inteligente

Sistema robusto desenvolvido para gerenciar operações de vendas em larga escala (Atacado), focando no controle rigoroso de inventário, movimentação de mercadorias e agilidade no fechamento de pedidos volumosos.

## 🚀 Acesse o Sistema
🔗 **Link do Projeto:** sistema-pdv-atacado.web.app

## 🛠️ Tecnologias e Arquitetura
* **Lógica de Negócio:** JavaScript (ES6+) processando cálculos de grandes volumes.
* **Persistência de Dados:** Firebase Realtime Database (Sincronização instantânea entre estoque e vendas).
* **Interface:** HTML5 e CSS3 com foco em produtividade e alta visibilidade de dados.
* **Infraestrutura:** Deploy via GitHub Pages com integração de banco de dados em nuvem.

## ✨ Funcionalidades Principais
* **Gestão de Estoque Inteligente:** Monitoramento de níveis de produtos com alertas de reposição.
* **Controle de Vendas em Atacado:** Interface otimizada para lançamento de grandes quantidades de itens.
* **Atualização em Tempo Real:** Sincronização imediata entre múltiplos terminais (vendas e almoxarifado).
* **Histórico de Movimentação:** Registro detalhado de entradas e saídas para auditoria de estoque.

## 📈 Diferenciais Técnicos
Este projeto foi arquitetado para suportar uma carga de dados superior a sistemas de varejo comuns, utilizando técnicas de otimização no Firebase para garantir que a consulta de grandes inventários seja instantânea, mesmo em conexões limitadas.
